# 手势控制星空网页

一个创新的交互式网页项目，通过手势识别技术实现与星空的互动。

## 功能特性

- **手势聚散控制**：通过摄像头捕捉手势，双指捏合控制星星的聚拢与分散
- **手掌旋转控制**：旋转手掌可以控制星空的旋转方向
- **动态星空**：1200 颗闪烁的星星，营造沉浸式太空体验
- **实时反馈**：左下角显示当前手势状态

## 使用方法

1. 在浏览器中打开网页
2. 点击「开启摄像头」按钮并授权摄像头访问
3. 对着摄像头做出手势：
   - **双指捏合**：星星聚拢
   - **双指张开**：星星分散
   - **旋转手掌**：控制星空旋转方向

## 技术栈

- HTML5 Canvas
- MediaPipe Hands（手势识别）
- 原生 JavaScript

## 本地运行

直接在浏览器中打开 `index.html` 文件即可。

## 注意事项

- 需要摄像头权限
- 建议在光线充足的环境下使用
- 支持桌面浏览器（Chrome、Firefox、Edge）
